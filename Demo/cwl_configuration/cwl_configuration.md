# VSCode环境配置

## 环境变量

环境变量是操作系统维护的一套全局公共键值对配置，相当于系统给所有运行的程序（包括你写的 exe、微信、浏览器等）准备的「公共共享记事本」。
其中「键」是变量名，「值」是对应的配置内容，所有程序都能随时读取，部分场景下也可修改。
因此需要添加Python解释器和MinGW到环境变量中，以在VSCode中运行代码时可以找到解释器和编译器。
